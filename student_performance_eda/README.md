# Student Performance EDA

Exploratory Data Analysis project using the UCI Student Performance Dataset.

## Goals
- Analyze factors associated with student grades
- Explore social and academic variables
- Practice pandas-based data analysis and visualization

## Technologies Used
- Python
- Pandas
- Matplotlib

## Key Findings
- G2 strongly predicts final grades
- Previous failures negatively impact performance
- Students choosing schools for reputation performed better on average
- Teacher-parent households showed slightly higher average grades
- Romantic relationships affected genders differently

## Skills Demonstrated
- Data cleaning
- Feature engineering
- Correlation analysis
- Groupby operations
- Data visualization
- Exploratory Data Analysis (EDA)
